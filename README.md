# Automated Reporting Pipeline

## Descripción

Proyecto desarrollado en Python para automatizar el procesamiento de datos empresariales.

El sistema carga múltiples archivos CSV, realiza limpieza de datos, genera métricas de negocio, crea visualizaciones y produce reportes ejecutivos en PDF.

## Funcionalidades

* Carga automática de múltiples archivos CSV
* Limpieza y validación de datos
* Cálculo de ingresos
* Análisis por producto
* Análisis por país
* Análisis de clientes
* Tendencias temporales
* Generación de gráficos
* Exportación a PDF
* Sistema de logs

## Tecnologías

* Python
* Pandas
* Matplotlib
* Seaborn
* ReportLab

## Flujo de trabajo

Clásico flujo de ETL
CSV → Limpieza → Análisis → Gráficas → Reporte PDF

## Resultados

El sistema genera automáticamente:

* Reporte PDF
* Reporte TXT
* Gráficas PNG
* Archivo de logs
